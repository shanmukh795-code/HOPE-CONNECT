## HOPE CONNECT 

HOPE CONNECT is a donation platform connecting donors with NGOs. This repository contains both the client (React) and server (Node/Express).

##  Quick Start

The project is currently running! But here is how to run it from scratch:

### 1. Prerequisites
*   Node.js installed on your computer.

### 2. Installation
You need to install dependencies for **both** the client and server.

**Server:**
```bash
cd server
npm install
```

**Client:**
```bash
cd client
npm install
```

### 3. Running the App
You need two terminals open (one for each part).

**Terminal 1 (Server):**
```bash
cd server
npm run dev
```
*   Server runs on: [http://localhost:3000](http://localhost:3000)

**Terminal 2 (Client):**
```bash
cd client
npm run dev
```
*   Client runs on: [http://localhost:5173](http://localhost:5173)

## Project Structure
*   `client/`: Frontend React application.
*   `server/`: Backend Express API and Database logic.
